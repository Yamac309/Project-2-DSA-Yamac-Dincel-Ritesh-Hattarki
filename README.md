Project Description: 
            - This project compares the performance of Merge Sort and Quick Sort on a large dataset. This program generates 100,000 flight records and stores 
              them in a vector. Then it sorts the data using both algorithms and measures the time taken for each.

Files:
            - main.cpp
            - data.cpp
            - mergesort.cpp
            - quicksort.cpp

How to Compile and Run:
            - Copy and paste this into the terminal: g++ -std=c++11 main.cpp data.cpp mergesort.cpp quicksort.cpp -o main
            after doing that. Paste "./main" into the terminal.

How to Use:
            - Choose option 1 to load the dataset
            - Choose option 2 to run Merge Sort
            - Choose option 3 to run Quick Sort
            - Choose option 4 to compare performance
            - Choose option 5 to exit
